# How to Open Project Tracker

Follow these steps one at a time. It's easy!

## 1. Download it
Click the download link. You'll get a file called **ProjectTracker.zip**.
Wait for it to finish downloading. It usually goes into your **Downloads** folder.

## 2. Unzip it
Find **ProjectTracker.zip** in your Downloads folder.
Right-click it and choose **Extract All**, then click **Extract**.
This turns the zip into a normal folder called **ProjectTracker**.

## 3. Open the app
Open the **ProjectTracker** folder.
Find the file called **ProjectTracker**, and **double-click** it.

## 4. If Windows shows a blue warning
The first time, Windows might say **"Windows protected your PC."**
Don't worry -- this is normal for brand-new apps and Project Tracker is safe.

- Click **More info**
- Then click **Run anyway**

You only have to do this the first time.

## 5. The app opens
A window opens with Project Tracker. The first time it may take a few seconds
to get ready. That's normal.

## 6. Log in
Type in these details:

- **Email:** admin@example.com
- **Password:** admin123

Then click **Log in**. You're in!

*(You can change this password later inside the app, in Settings.)*

## 7. Pin it so it's easy to find next time
While Project Tracker is open, look at the bar along the bottom of your screen.

- **Right-click** the Project Tracker icon down there
- Choose **Pin to taskbar**

To add it to your Start menu too:

- **Right-click** the **ProjectTracker** app file
- Choose **Show more options** (if you see it), then **Pin to Start**

Now you can open Project Tracker any time with one click. Done!
